# rikki is now nevla

This project was renamed. The language, its binaries, and its extension
are now nevla / `nv` / `.nv`, published as
[`nevla`](https://pypi.org/project/nevla/).

```sh
uv tool install nevla    # or: pip install nevla
```

The old name came from Kipling's Rikki-Tikki-Tavi, whose imperial
allegory the project would not carry. The full account is in
[ADR 0014](https://github.com/guygrigsby/nevla/blob/main/docs/adr/0014-rename-to-nevla.md).

Site: https://nevla.dev
